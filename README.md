KeenKeeper
Description

KeenKeeper is a responsive web application that helps users maintain and track their friendships by logging interactions and monitoring relationship status in a simple and intuitive way.

Technologies Used
React.js
React Router DOM
Tailwind CSS and DaisyUI
Recharts
JavaScript (ES6)
Key Features
Track friendship status (Overdue, Almost Due, On Track)
Quick interaction logging (Call, Text, Video) with timeline updates
Visual analytics using Pie Chart
