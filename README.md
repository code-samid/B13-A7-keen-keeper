# KeenKeeper

## Description  
- KeenKeeper is a responsive web application  
- Helps users maintain and track friendships  
- Logs interactions and monitors relationship status  
- Simple and intuitive user interface  

## Technologies Used  
- React.js  
- React Router DOM  
- Tailwind CSS and DaisyUI  
- Recharts  
- JavaScript (ES6)  

## Key Features  
- Track friendship status (Overdue, Almost Due, On Track)  
- Quick interaction logging (Call, Text, Video) with timeline updates  
- Visual analytics using Pie Chart
